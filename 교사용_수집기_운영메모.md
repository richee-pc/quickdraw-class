# 교사용 수집기 운영 메모 (학생 앱 분리용)

이 문서는 학생 배포용 `app.py` 에 넣지 않는 교사용 운영 정보만 모아둔 메모입니다.

## 1) 자동 저장 엔드포인트

- 현재 설정 위치: `나만의_퀵드로우_수집기.html`
- 설정 변수:
  - `AUTO_SAVE_URL`: Apps Script Web App `.../exec` URL
  - `AUTO_SAVE_CLASS_ID`: 수업 식별자

학생 화면에는 해당 설정 UI가 노출되지 않습니다.

## 2) 하루 캠프 운영 팁

- 학생에게는 **이름만 입력**하도록 안내
- 물체는 한 가지(예: `car`)로 통일
- 초반에는 "자유롭게 그려보라"고만 안내
- 이후 오인식 사례가 나오면 "어떤 모양이 부족했는지"를 토의

## 3) 오류 점검

- 저장이 안 되는 경우:
  1. `AUTO_SAVE_URL` 값 오타 확인
  2. Apps Script 웹앱 배포 권한 확인(링크 사용자 접근 가능)
  3. 브라우저 콘솔의 네트워크 오류 확인
