# 교사용 가이드 — 파이썬 기반 커스텀 퀵 드로우 만들기

> **이 문서는 선생님이 수업 전에 먼저 읽는 자료입니다.** 처음 접하셔도 괜찮도록, 개념 → 사전 준비 → 진행 대본 → 문제 해결 순으로 정리했습니다. 수업 자체가 처음이어도 이 문서대로 따라가면 진행할 수 있습니다.

---

## 0. 한 줄 요약

> **"미드저니로 게임의 옷을 입히고(생성형 AI), 퀵 드로우로 게임의 두뇌를 만든다(분류 AI)."**

학생들은 6시간 동안 **그림을 알아맞히는 AI를 직접 만들고**, **펜마우스로 그려서 자기 AI와 게임**합니다. 미드저니는 게임의 아트(로고·캐릭터·배경)를 만드는 데 씁니다.

---

## 1. 선생님이 먼저 이해할 핵심 개념

### (1) AI 비전의 두 얼굴

| 구분 | 생성형 AI (만든다) | 분류 AI (알아맞힌다) |
|------|------------------|--------------------|
| 예시 | **미드저니** (글 → 그림) | **퀵 드로우** (그림 → 이름) |
| 하는 일 | 텍스트로 새 이미지를 창작 | 그림을 보고 무엇인지 판단 |
| 수업 역할 | 게임 아트 제작 | 우리가 직접 만드는 핵심 결과물 |

### (2) 분류 AI가 작동하는 4단계 (학생에게도 이 4단어로 설명)

1. **데이터(Data)** — 사람들이 그린 '고양이' 낙서 수천 장
2. **학습(Training)** — 컴퓨터가 그림들의 공통 패턴을 스스로 익힘
3. **모델(Model)** — 학습이 끝나 똑똑해진 '두뇌'
4. **추론(Inference)** — 새 그림을 보여주고 "이게 뭐게?" 물어봄 (= 펜마우스로 그리는 순간)

### (3) 더 깊이 알고 싶을 때 (선택)

- **CNN(합성곱 신경망)**: "그림 속 특징(선 → 모서리 → 모양)을 단계적으로 잡아내는 두뇌 구조"라고만 이해하셔도 충분합니다. 노트북에서 `Conv2D` 가 그 역할을 합니다.
- **정확도(accuracy)**: 시험 그림 100장 중 몇 장을 맞혔는지의 비율. 학습이 잘 될수록 올라갑니다.
- **데이터 출처**: 구글 Quick, Draw! 게임에서 전 세계 사람들이 그린 낙서를 모은 공개 데이터셋입니다.

- [ ] 이 폴더의 `커스텀_퀵드로우.ipynb` 를 **선생님이 미리 한 번 끝까지 실행**해 보기 (가장 중요!)

### 선생님이 노트북을 미리 돌려보는 법 (필수 리허설)

2. `파일 → 노트북 업로드` → 이 폴더의 `커스텀_퀵드로우.ipynb` 선택

5. 끝까지 한 번 성공해 보면, 당일 어떤 질문이 와도 대응할 수 있습니다.

> ⏱️ 전체 실행은 보통 5~10분이면 끝납니다(인터넷 속도에 따라 데이터 내려받기 시간 차이).

## 3. 미드저니 사용 팁 (베이직 플랜)

- **프롬프트 기본 공식**: `주제 + 스타일 + 색감/분위기`
  - 예) `cute cat mascot, flat vector illustration, pastel colors, simple` (귀여운 고양이 마스코트, 플랫 벡터, 파스텔)
- **베이직 플랜 주의**: Fast(빠른 생성) 시간이 제한적입니다. 학생들에게 **프롬프트를 먼저 종이에 적게** 한 뒤, 꼭 필요한 **4~6장만** 생성하도록 지도하세요.
- 학생 1인당 만들 것: **① 게임 제목 로고 ② 정답 캐릭터(마스코트) ③ 배경** — 마음에 드는 것만 다운로드 저장.
- 퀵 드로우 카테고리와 캐릭터를 맞추면 좋습니다(예: cat 카테고리 → 고양이 마스코트).

## 6. 자주 나오는 문제 & 해결 (FAQ)

| 증상 | 원인 / 해결 |
|------|-------------|
| 셀 실행이 안 됨 | 위 셀부터 **순서대로** 실행했는지 확인. `런타임 → 모두 실행` 으로 한 번에 가능 |
| `name 'model' is not defined` | 5단계(모델 학습)를 건너뜀 → 다시 5단계 실행 |
| 데이터 내려받기 실패 | 인터넷/학교망 문제. 카테고리 영어 철자 확인 (사이트 목록 참고) |
| 그림판 링크가 안 열림 | 링크 두 개 중 `https://....gradio.live` 링크 사용. 잠시 후 다시 시도 |
| AI가 자꾸 틀림 | ① 그림을 크고 또렷하게 가운데에 ② `EPOCHS`·`N_PER_CLASS` 키우기 ③ 카테고리 수 줄이기 |
| 세션이 끊김 | Colab 무료는 일정 시간 후 끊김. 다시 `런타임 → 모두 실행` |
